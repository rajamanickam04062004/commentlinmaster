Security and Permissions Commands

1.attrib

Changes file attributes (read-only, hidden)

attrib +h file.txt

2.cacls

Changes file permissions

cacls file.txt /G username:F

3.cd..

Moves to the parent directory.

cd..

Creates a new folder.
