Utility and Debugging Commands

1.cls

Clears the Command Prompt screen

cls

2.echo

Prints a message on the screen or writes text to a file

echo Hello World

3.pause

Pauses the execution of a script or batch file

pause

4.set

Displays, sets, or removes environment variables

set PATH

5.fc

Compares two files and shows the differences

fc file1.txt file2.txt
